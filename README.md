#Migration Scripts (Tenant-to-Tenant Toolkit)

Here you will find a collection of PowerShell scripts and utilities which can be helpful in tenant-to-tenant (T2T) migration projects.

Includes scripts for
- Exchange Online
- SharePoint Online
- OneDrive for Business
- Active Directory / Entra / Identity
- ShareGate support

Some scripts are intended to support workflows using ShareGate (a migration tool for SharePoint, Teams, and OneDrive).
Learn more: 
- https://www.sharegate.com
